Do we have final project ideas for a PBL Web Project?

Game: Tetris, other block games

Are we considering a project that is best for our educational purpose?

We are shooting for a game that enhances the brain and boosts IQ. Tetris enhances our brain's spatial and visual system, it creates better memory and efficiency, and can even relieve stress and ease anxiety.

Is the project going to hold team members interest for 8 weeks?

Yes, we are pretty interested in coding our own game. We are open and ready to improve on our skills, face challenges and overcome them on the way. Figuring out a problem in coding boosts our morale and motivation to continue coding.

Does the project have potential for someone to use it beyond the 8 weeks? ie Customer or Sponsor needed?

Yes, many people play games over a long period of time. It can be used during free time or breaks. There is not a customer or sponsor needed. It is a coding assignment simply on VSCode.

Does the project have potential to be used for Create Performance Task submission?

Maybe, some believe that this game is too basic and not as creative as it could be. Maybe if we tweak and change the game a little it could qualify more for the Create Performance Task.

performance task criteria:

Row 1: Program Purpose and Function: our project's functionality can be demonstrated ina successful playthrough of the game.

Row 2: Data Abstraction: we will need to store data if we want a point system, and then a way to display that data.

Row 3: Managing Complexity: the project will have complexity as there are a lot of different parts. we will need to manage this with strategies like breaking into code segments or lists to manage complexity.

Row 4: Procedural Abstraction: procedures will be needed to reduce complexity of having a bunch of code run at once for our project.

Row 5: Algorithm Implementation: we will need algorithms for things like randomly selecting blocks or processing info in game.

Row 6: Testing: We can test the game by playing through and seeing if it works. investigation will be looking at other's tetris projects and quoting their work.
